# Mi proyecto web
Proyecto: Guía de supervivencia; Tecnicatura en Programación.

# Descripción
Esta página se trata de compartir una serie de consejos para facilitar la gestión y organización de los tiempos de estudio. Brinda al espectador aplicaciones útiles, información para iniciarse en la programación, herramientas para facilitar el aprendizaje, tips para evitar dolores físicos y una serie de pasos que puede seguir en situaciones donde el código no coopera con el tiempo. También cuenta con un formulario aparte para conocer otros consejos y saber la opinión del espectador respecto a la web.

# Contenido
Menú principal en donde enlazamos hacia las diferentes secciones que hay en la página.

Primera sección: Donde vemos el título del tema a tratar. Cuenta con una frase introductoria, una breve descripción de la guía, una imagen y, al final de la sección, cuenta con un link para volver al inicio.

Segunda sección: Donde visualizamos el título y se empieza a hablar de la gestión y organización de los tiempos de estudio. Después de una introducción, se muestra una serie de consejos a tener en cuenta. También contamos con el primer -tip- y, finalizando la sección, un link para volver al inicio.

Tercera sección: Muestra el título "Apps salva-cuatrimestres", donde nos da una corta introducción para pasar a la enumeración de las aplicaciones recomendadas. Al final de la sección tenemos un video de recomendación para principiantes y, debajo, un link para volver al inicio. Bajo el link finalizamos esta sección con un espacio publicitario.

Cuarta sección: Tenemos el tema "Comparativa de editores de código", que nos deja una presentación del tema. Luego tenemos la tabla comparativa, debajo de esta un -tip- y para concluir un link para volver al inicio.

Quinta sección: Empezamos a abordar un "Protocolo de emergencia" que propone una serie de pasos a seguir cuando la persona no encuentra un error o se frustra de más en alguna parte del código. Terminamos esta sección con un link para volver al inicio.

Sexta y última sección: Donde se invita al espectador a comentar sus consejos y deje su opinión acerca de la página, dándole un link para ingresar y completar el formulario. Al final tenemos un -tip- seguido de un "espacio publicitario".

En el pie de página tenemos un último link para dirigirnos al comienzo. Debajo, información acerca de la creación de la página y su autora.
