Apps salva-cuatrimestres

Acá tenemos un enlistado de Apps con las cuales se puede contar cuando estamos pasando por tiempos oscuros durante el aprendizaje o simplemente para seguir aprendiendo cada día un poco más

  1. Notion (La central de mando)

    • Es la herramienta definitiva para organizar materias. Permite crear páginas con bloques de código que respetan los colores de la sintaxis, lo que facilita muchísimo repasar antes de un examen sin tener que abrir el editor.

  2. Obsidian

    • A diferencia de otras apps, esta funciona mediante nodos. Es perfecta para conectar conceptos de programación: podés vincular una nota de "Lógica" con una de "C#" y ver visualmente cómo se relaciona todo lo que aprendés.

  3. Anki (Flashcards de memoria)

    • Utiliza la repetición espaciada para que no te olvides de los comandos de SQL o de Git. Creás tarjetas con preguntas y respuestas, y la app te las muestra justo cuando estás por olvidarlas. Es magia para los finales teóricos.

  4. Mimo / SoloLearn

    • Ideal para aprovechar los tiempos muertos, como cuando viajás en el bondi. Te permite practicar pequeños desafíos de HTML, CSS o Python en formato de juego, manteniendo el cerebro ágil sin que se sienta como una tarea pesada.

  5. Excalidraw (Pizarrón digital)

    • Antes de escribir una sola línea de código, hay que pensar. Esta app te permite dibujar diagramas de flujo y esquemas de bases de datos con una estética de dibujo a mano que hace que planificar sea mucho más divertido.
